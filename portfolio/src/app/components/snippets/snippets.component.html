But really... expand_more expand_less

My passion lies in back end development and creating microservices.

I also enjoy creating custom eye catching and intuitive user interfaces. I prefer using Angular for UI development with Bootstrap and Material design to insure responsive and accessible front-ends.

Responsive design expand_more expand_less

In 2017, over 50% of the use of the web was from mobile devices. It is my firm position that non-responsive applications are the mark of a lazy developer.

I use Bootstrap or create custom responsive frameworks for my UI work. While in the enterprise world, I understand that are justifications for non-responsive sites in large enterprise applications, however, I see this need as a dying flame.

I've got your back expand_more expand_less

I never treat application security as an afterthought. I always place a very large emphasis on keeping data safe.

I use security frameworks that are mature and have been tested over time. For authentication and authorizaion, I most enjoy using the .NET Identity Framework as it is versatile, supports Single Sign On and is extensible and customizable.
